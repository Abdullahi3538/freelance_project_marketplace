# Freelance Project Marketplace

A full-stack web application that connects clients and freelancers. Clients can post projects with budgets and deadlines, while freelancers can create profiles, showcase their skills, submit bids, and work through milestones until project completion.

## 📌 Project Overview

The Freelance Project Marketplace is designed to simplify the hiring process between clients and freelancers. The platform allows clients to publish projects and freelancers to compete by submitting proposals and bids. Once a bid is accepted, a contract is created, milestones are tracked, payments are released, and both parties can leave ratings and reviews.

## ✨ Features

### Client Features

* User registration and authentication
* Create, update, and delete projects
* Set project budgets and deadlines
* View and compare freelancer bids
* Award contracts to freelancers
* Track project milestones
* Release milestone payments
* Rate and review freelancers

### Freelancer Features

* User registration and authentication
* Create a professional profile
* Add and manage skill tags
* Browse available projects
* Submit bids and proposals
* Track contract progress and milestones
* Receive payments for completed milestones
* Rate and review clients

### Admin Features

* Manage users
* Monitor projects and contracts
* Handle system administration tasks

## 🗄️ Database Design

The system consists of the following tables:

1. Users
2. Freelancer Profiles
3. Skills
4. Freelancer Skills
5. Projects
6. Bids
7. Contracts
8. Milestones
9. Reviews
